Basic HTML Text Tags

Beginning practice with the basic text formatting tags used in HTML5.

Comments and Text Blocks

Comments are used to add additional information to pages that helps developers working on that code. You can also use a comment to hide content from your page.

Marking up text in blocks usually involves using paragraph tags to contain multi-line text. These paragraphs can include emphasized words, highlighted text, and other inline elements.

It may happen that you have text, like a poem, that requires special formatting. In that case, you may use line break tags. Below is an example from The Raven by Edgar Allen Poe:

Excerpt from The Raven

Once upon a midnight dreary, while I pondered, weak and weary,
Over many a quaint and curious volume of forgotten lore—
    While I nodded, nearly napping, suddenly there came a tapping,
As of some one gently rapping, rapping at my chamber door.
“'Tis some visitor,” I muttered, “tapping at my chamber door—
    Only this and nothing more."
